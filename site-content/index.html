Step 1: Planning

Before you start building your website, it’s important to have a clear plan in mind. Think about the purpose of your website, who your target audience is, and what type of content you want to include.

Step 2: Creating the HTML Structure

Start by creating the basic structure of your website using HTML tags. Use h1, h2, and h3 tags to create headings and subheadings that will help organize your content.

Step 3: Styling with CSS

Once you have the basic structure in place, use CSS to style your website and make it visually appealing. You can add background images, colors, fonts, and other design elements to enhance the look of your site.

Step 4: Put your hands on the Code

With the Applications on your host(docker,docker-compose), lets get started running only two commands
git clone https://github.com/leonardodutra/cloudsolvd.com.git; docker-compose up -d

Step 5: Conclusion

You have a complete docker deployment with nginx running you application in tcp port localhost:80
